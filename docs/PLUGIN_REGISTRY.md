# Community Plugin Registry
Current version: 3.1.0


This page lists community-maintained plugins and integrations for JTCSV.

## Official
- `@jtcsv/express-middleware`
- `@jtcsv/fastify`
- `@jtcsv/nextjs`
- `@jtcsv/nestjs`
- `@jtcsv/remix`
- `@jtcsv/nuxt`
- `@jtcsv/sveltekit`
- `@jtcsv/hono`
- `@jtcsv/trpc`

## Community
Add your plugin here by opening a PR with:
- Package name and npm link
- краткое описание (1–2 строки)
- Supported JTCSV version range

## Submission checklist
- MIT/Apache-2.0 compatible license
- Clear README usage example
- CI/test status badge (optional)
